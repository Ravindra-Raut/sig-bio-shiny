# Help
***

## Input 
Input in text area must be **ENSEMBL** gene ID.

If you providing fold change (which is **optional**, required for certain plots), include them **separate by a coma**.

Example (GeneIDs with fold change)
```
ENSG00000012048,4.6
ENSG00000214049,-3.7
ENSG00000204682,2.5
```
